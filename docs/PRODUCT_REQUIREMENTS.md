# Product Requirements

## Finance Module

### Features
- Add expense
- Add income
- Expense categories
- Budget tracking
- Dashboard summary

### Rules
- Expenses must belong to a category
- Amount cannot be negative
- Users can only access their own data

---

## Habit Module

### Features
- Create habits
- Daily tracking
- Streak tracking
- Weekly completion %

### Rules
- Habits belong to users
- One log per day per habit

# Life OS

## Overview

Life OS is a personal productivity and finance management application.

The application currently focuses on:

- Personal finance tracking
- Habit tracking
- Authentication and user management

Future scope may include:
- Health tracking
- AI assistant
- Notifications
- Analytics

---

## Tech Stack

### Frontend
- React
- TypeScript
- Vite
- TailwindCSS
- Zustand
- TanStack Query

### Backend
- FastAPI
- MongoDB
- Motor
- JWT Authentication

---

## Architecture

This project follows a modular monolith architecture.

Backend follows:
Route → Service → Repository → Database

Frontend follows:
Feature-based architecture.
